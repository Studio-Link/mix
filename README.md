# Studio Link - Mix

Next generation decentralized social network based on voice/video and broadcasting support.


## Roadmap

- OAuth2 support
- ActivityPub
- WebRTC
- RTMP Broadcasting
- Server/Client Recording

## Development

Compile and start backend

```sh
$ make
$ build/slmix
```

Start frontend

```sh
$ cd webui
$ npm install
$ npm run dev
```
